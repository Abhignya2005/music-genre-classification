# Music Genre Classification using PyTorch

## Overview
This project classifies music into genres like Classical, Jazz, Pop, and Rock using a neural network built with PyTorch.

## Features
- Music genre prediction
- Audio preprocessing and normalization
- Neural network model using PyTorch
- Simple interface to upload audio

## Tech Stack
Python, PyTorch, NumPy, Pandas, Matplotlib, HTML, CSS

## Project Structure
app.py - Main file to run the project  
ANNmodel.py - Model definition  
audioprocessing.py - Audio feature extraction  
templates/ - HTML files  
static/ - CSS and assets  

## How to Run
1. Clone the repository  
2. Install requirements  
3. Run: python app.py  

## Note
Dataset and model files are not uploaded due to size limits.
