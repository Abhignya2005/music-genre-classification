we can write Script tag in both head and body but better to write in body 

for validation purpose we are writing the JavaScript 

everything is treated as object in JS 

Document is an object which point to the webpage 

----------> The DOM is a platform (independent ) and language - neutral interface that allows programs and scripts to dynamically access and update the content , structure(want to add or append or remove elements into it ), and style(backgraound color , margins etc..) of a document

2 types > DOM --- HTML and XML


Why DOM ? 
-> to access all elements of the HTML page 
-> to change the content of the elements of the HTML page 
-> to apply, change and delete the CSS properties of the HTML page 
-> to add the EVENTS(change in the behaviour) to the HTML elements 
-> to make changes according to the EVENTS occurrence

DOM Method

1) document.getElementById("id") ---------> return element value based on id value declared in the document

2) document.getElementByTagName("tagname") ---------> return list of all elements based on tag name

3) document.getElementByClassName("classname") --> returns list or collections of elements
